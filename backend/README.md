# ReactAppV3

Modular Asset Management System

## Quick Start

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run init-db
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Default Login
- Username: `admin`
- Password: `admin123`
